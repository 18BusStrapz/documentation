# Related Links, External Resources & Reference Materials
More and more developer resources live online nowadays. Saving the links you found while researching or problem-solving in the browser, code, or documentation has never been easier.

With <a href="{{ links.website.home }}">Pieces for Developers</a>, you can quickly associate URLs to external resources like Documentation Pages, Wiki Links, Jira Tickets, Pull Requests, and <a href="{{ links.internal.share_via_github_gist }}">GitHub Issues </a>with your saved materials.
