## Having Trouble with Pieces for Obsidian?

If the Obsidian plugin isn't working for you, please make sure you have the following:

- The latest version of Pieces for Obsidian
- Pieces OS version 1.9.2 or higher
After installing or updating your software, please restart Obsidian. If there is still a problem, check out the frequently asked questions below.
