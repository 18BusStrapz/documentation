### FAQ: How does "Sort by Suggested" differ from "Sort by Last Added"?

#### ### The Pieces for Developers desktop app provides users with two options to sort the snippet carousel:

1. Sort by **Suggested**
- This will resurface previously saved Snippets and reorder the carousel to put relevant snippets closer the beginning of the carousel.
- It's useful for quickly accessing relevant snippets from your current workflow.
2. Sort by **Last Added**
- focused on the recency of the snippets
- useful for when users want to stay up to date with the latest Snippet creations

### FAQ: How do I Export a Gist to GitHub?

1. Generate a shareable link
2. Inside this menu, identify the GitHub icon "Share as GitHub Gist"
3. Select this icon
4. You will see a GitHub Menu
5. You can optionally give your snippet a custom Name & Description
6. Click "Create" to share to GitHub

Notes:
- Gists can be set to private in the share a Gist menu

### FAQ: How do I Import a Gist to the Pieces for Developers Desktop App?

**Prerequisites to Import from GitHub:**
1. Connect to Pieces OS
2. Sign in via personal email or google account
3. Authenticate with GitHub
4. Sign in account must be linked with GitHub account
_mismatched accounts will fail when attempting to Import from GitHub._


**Steps to Perform a Gist Import**
1. Choose the “Discover” option above your Snippet Carousel
2. Choose the option “Select Gists”
3. Make sure Pieces is connected to your GitHub account
4. Browse your Gists in the Desktop App and select the ones you’d like to import
5. Done!

This feature represents a major step forward in our mission to provide users with the most seamless and intuitive coding experience possible. Whether you're a seasoned developer or just starting out, the ability to import your existing GitHub Gists is sure to make your life as a developer easier.

### FAQ: If I delete a Gist in GitHub will the snippet also be deleted in the Pieces Desktop App?

####
- Deleted Gists will not be deleted in Pieces for Developers.

### FAQ: I deleted a Gist from GitHub. Can I recover the original Gist?

**Yes**, a Gist can be recovered from the original snippet in Pieces.

To Redo a Gist:

1. Locate the desired snippet in the Pieces Carousel
2. Navigate to the shareable link within the snippet's information view
3. Click the GitHub icon to view the "share as GitHub Gist" menu
4. Select the "redo Gist" icon
5. This will repopulate your GitHub Gist list with the selected snippet

### FAQ: **What is Pieces OS? How does it work?**
#### Pieces OS is the background service that enables Pieces to work locally and handles communications with any integrations. Without Pieces OS, Pieces for Developers will not be able to save your snippets or interact with any of your plugins.
#### Pieces OS also enables you to sync Pieces to the cloud so that you can access your data anywhere. It also enables our other cloud-enabled features, like link sharing.

Head over to https://code.pieces.app/install to get started.


### FAQ: I'm Disconnected from Pieces OS. Now what?
#### If you've recognized that you're unable to connect to Pieces OS, here's what to do:
1. Navigate to your applications folder from the finder window
2. Locate Pieces OS & open the application

- If you merely closed the app on accident, the steps above should resolve your connection issue.

#### Still not Connected?

- You may need to install Pieces OS.

Head over to https://code.pieces.app/install to get started.

#### Still not Connected?

- If you're still disconnected after installing Pieces OS, check your OS compatability.

### FAQ: How do I convert a saved Image to Text?

1. Open the Pieces for Developers Desktop App.
2. In the snippet carousel, identify the image you want to extract text from and focus on it.
3. Click on the "View as Code" button.
4. The app uses OCR technology to detect and extract the text from the image, which will be displayed on the screen.
5. You can now copy, modify, and share the extracted text as needed.

_It's important to note that the accuracy of the extracted text may vary depending on the quality of the image and the complexity of the text._

#### The app can analyze screenshots and convert them to editable text, which can save time and effort by eliminating the need to manually retype the text.

### FAQ: What are the key benefits of the Pieces for GitHub Integration?


#### 1. Enhanced Collaboration:
By automatically capturing relevant Git history, including collaborator profiles, commit messages, and related pull request and branch links, Pieces enables users to search their snippets via collaborator profiles, making it easier than ever to know who to reach out to or add to a pull request, ultimately enhancing collaboration.

#### 2. Fully Configurable:
The context captured by the new Git Context feature is fully editable, searchable, and shareable, giving users complete control over their snippets and making it easy to find and reuse code. Additionally, all the context information is configurable in the Pieces Settings, allowing users to customize their experience to best fit their needs.


#### 3. Context Awareness:
Pieces for Developers makes it easy for developers to save from their respective IDE and allow the Pieces for GitHub integration to automatically include a list of people who made commits to the code, enabling them to understand who wrote and interacted with the code. This feature is particularly useful for teams who want to collaborate more effectively.

### FAQ: Where do the suggested searches inside the Global Search view come from?
The suggested searches in the Global Search view are made up of existing snippet tags from the user's personal repository. When a suggested search is clicked, users are directed to a list view of snippet names, titles, classifications, and commonly shared related tags and links. This powerful feature streamlines the search experience in Pieces for Developers and allows users to discover relevant code snippets with ease.
