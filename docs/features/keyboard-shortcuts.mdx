    <TabItem value="windows/linux">
        | Add a Piece | keybind |
        |---|---|
        | Add code snippet to Pieces |  ` CTRL + V ` |
        | Add code file to Pieces | ` CTRL + U ` |
        | Create snippet from scratch | ` CTRL + N ` |
        | Add collections to Pieces | ` CTRL + SHIFT + C ` |

        | Manage a Piece | keybind |
        |---|---|
        | Rename |  ` CTRL +  R ` |
        | Delete | ` DELETE ` |
        | Open Quick Menu | ` CTRL +  O ` |
        | Reclassify | ` CTRL +  Y ` |
        | Edit Piece Description |  ` CTRL + SHIFT + N ` |
        | Add Related Links | ` CTRL +  K ` |
        | Add Tags | ` CTRL +  T ` |
        | Toggle Information View | ` CTRL +  I ` |
        | Open Actions Menu |  ` CTRL +  ENTER ` |
        | Highlight Sensitive Information | ` CTRL +  ! ` |

        | Share with People | keybind |
        |---|---|
        | Generate Sharable Link |  ` CTRL +  L ` |

        | Use a Piece | keybind |
        |---|---|
        | Paste code snippet into IDE |  ` CTRL +  V ` |
        | Duplicated a Piece | ` CTRL +  D ` |
        | Download a Piece | ` CTRL + SHIFT + D ` |
        | Copy with Context | ` CTRL + ALT +  C ` |

        | Editing a Piece | keybind |
        |---|---|
        | Focus Editor |  ` CTRL +  E ` |
        | Save Edits | ` CTRL +  S ` (while editing) |
        | Escape Editing | ` ESC ` |

        | Navigating Pieces | keybind |
        |---|---|
        | Workflow Activity View |  ` CTRL + SHIFT + A ` |
        | Focus Search | ` / ` or ` CTRL + F ` |
        | View Next or Previous Piece | ` ← ` or ` → ` |
        | Jump to First or Last Piece | ` CTRL + ← ` or ` CTRL + → ` |
        | Toggle between Image and Code View | ` T ` |

        | Configuring Pieces | keybind |
        |---|---|
        | Open Settings |  ` , ` |
        | Toggle Theme | ` CTRL + SHIFT +  T ` |
        | Support Center | ` CTRL +  ? ` |
        | Submit Feedback | ` SHIFT +  ! ` |
        | Check for Update | ` SHIFT +  U ` |
        | Add an Integration  |  ` CTRL +  I ` (in settings menu) |
        | Zoom In | ` CTRL +  + ` |
        | Zoom Out | ` CTRL +  - ` |
        | Refer a Friend | ` CTRL +  J ` |
        | Live Chat | ` SHIFT +  H ` |
        | Release Notes | ` SHIFT +  W ` |
        | View Keyboard Shortcuts | ` CTRL +  ; ` |


    </TabItem>
