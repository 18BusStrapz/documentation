To run Pieces for Developers on Linux, be sure to download both of the applications from the <a target="_blank" href="https://snapcraft.io">Snap Store</a> before installing. You can also follow the instructions below these install buttons to install via terminal.

<InstallButton src="https://snapcraft.io/pieces-for-developers">Pieces Desktop App on Snapcraft.io</InstallButton>
<InstallButton src="https://snapcraft.io/pieces-os">Pieces OS on Snapcraft.io</InstallButton>

## Steps to Install

1. To install from your terminal run the following:
    ```shell
    $ sudo snap install pieces-os
    ```
    [Save to Pieces](https://snippets.pieces.cloud/?p=8805498237)

2. Enable Offline and On-Device Local Machine Learning by running the following command in the terminal:
    ```shell
    $ sudo snap connect pieces-os:process-control :process-control
    ```
    [Save to Pieces](https://snippets.pieces.cloud/?p=b8a24c9390)

    This command sets up and configures local application interfaces to properly load our Sand-boxed and Secure Machine Learning Models.

3. Install the Pieces Desktop Snap:
    ```shell
    $ sudo snap install pieces-for-developers
    ```
    [Save to Pieces](https://snippets.pieces.cloud/?p=f23c48bd96)

4. Launch Pieces OS by running the following command in your terminal:
    ```shell
    $ pieces-os
    ```
    [Save to Pieces](https://snippets.pieces.cloud/?p=0f964fb0a1)

5. Launch the Pieces for Developers Desktop App by running the following command in your terminal:
    ```shell
    $ pieces-for-developers
    ```
    [Save to Pieces](https://snippets.pieces.cloud/?p=e66b498977)

Check out our <a target="_blank" href="https://www.youtube.com/@getpieces">YouTube Channel</a> for more information on Getting Started.

## Helpful Tips for Linux

### Updating your Application
To update your application, you can either:

Update individual snap packages one at a time
    1. `sudo snap refresh pieces-for-developers`
    2. `sudo snap refresh pieces-os`

#### OR

Update all snap packages at once:
    1. `sudo snap refresh`

     **Note**: this will update other snap packages that you have installed, so be sure before you run this command.
