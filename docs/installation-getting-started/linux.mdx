# Getting Started with Pieces for Developers on Linux
<ReleasePill>Compatible with Ubuntu 18 or higher and currently in Beta</ReleasePill>


## Steps to Install

### 1. From the Snapcraft Store, install the Pieces for Developers | OS Server Snap Package, or run the following in your terminal:

```shell
$ sudo snap install pieces-os
```

### 2. Enable Offline and On-Device Local Machine Learning by running the following command in the terminal:

```shell
$ sudo snap connect pieces-os:dotnet-runtime-aspnetcore
```

- This command sets up and configures local application interfaces to properly load our Sand-boxed and Secure Machine Learning Models.

### 3. Launch Pieces for Developers | OS Server by running the following command in your terminal:

```shell
$ pieces-os
```

### 4. Launch the <a href="{{ links.website.home }}">Pieces for Developers | Flagship Desktop App</a> by running the following command in your terminal:

```shell
$ pieces-for-developers
```
