# Getting Started with [Pieces for Developers]( {{links.website.home}} ) on macOS
> Compatible with <a href="{{ links.installation.macos_catalina }}">macOS 10.15 Catalina or higher</a>

## Steps to Install
1. Download the <a href="{{ links.installation.pieces_pkg }}">Pieces for Developers Package Installer</a> and proceed with the installation.

This will install the <a href="{{ links.website.pfd_desktop_install }}">Pieces for Developers | Flagship Desktop App</a> as well as **Pieces for Developers | OS Server**.

![Installing Pieces for Developers from the Pieces website.](https://storage.googleapis.com/pieces_static_resources/vs_code_marketplace/GIFs/INSTALL_MACOS)

2. Make sure **Pieces for Developers | OS Server** is running in the background.
3. Launch the <a href="{{ links.installation.localhost }}">Pieces for Developers | Desktop App</a>

If you've already installed the <a href="{{ links.website.home }}">Pieces for Developers | Desktop App</a> and it's having trouble connecting to Pieces for Developers | OS Server, try these quick <a href="{{ links.internal.faqs }}">Troubleshooting Solutions</a>.
