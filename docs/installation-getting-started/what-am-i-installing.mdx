# What am I installing?

![](/assets/what_am_installing.png)

<InstallButton>Install Pieces for Developers</InstallButton>

When you download Pieces for Developers from our website, you'll get two applications: Pieces for Developers | OS Server and the Pieces for Developers | Flagship Desktop App. 

### <a target="_blank" href="{{ links.website.pfd_desktop_install }}">Pieces for Developers | OS Server</a>

is our background service that runs offline and on-device. It's required to enable:
- Individual products in the Pieces for Developers Suite to communicate
- Local, Private, and Secure Machine Learning Models
- Real-Time Search and Suggestions

### <a target="_blank" href="{{ links.website.pfd_desktop_install }}">Pieces for Developers | Desktop App</a>

is our flagship desktop application. It's a visual GUI that gives you **the best** experience with your snippets allowing you to:
- Navigate snippets visually through the carousel
- Transform snippets from one language to another
- Sort Snippets by suggestions
- Describe a snippet to get a solution to an answer
- Manage your custom cloud domain
- and so much more!

#### <a target="_blank" href="{{ links.website.pfd_desktop_install }}">Pieces for Developers | Flagship Desktop App</a> is the _core Pieces for Developers experience_, fully loaded with 50+ features.
