# What am I installing?
When you download Pieces for Developers from our website, you'll get two applications: Pieces for Developers | OS Server and Pieces for Developers | Flagship Desktop App. 

<a href="{{ links.website.pfd_desktop_install }}">Pieces for Developers | OS Server</a> is our background service that runs offline and on-device. It's required to enable:
- Individual products in the Pieces for Developers Suite to communicate
- Local, Private, and Secure Machine Learning Models
- Real-Time Search and Suggestions

<a href="{{ links.website.pfd_desktop_install }}">Pieces for Developers | Flagship Desktop App</a> is the _core Pieces for Developers experience_, fully loaded with 50+ features.
