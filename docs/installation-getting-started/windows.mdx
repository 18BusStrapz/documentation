## Steps to Install
1. Download the Pieces for Developers <a href="{{ links.installation.pieces_suite_manager }}">Installation Manager</a> and proceed with the installation.

    - This will install the Pieces for Developers | Desktop App as well as Pieces for Developers | OS Server.

2. Make sure **Pieces for Developers | OS Server** is running in the background.

    - Following the installation, ensure that the Pieces logo appears in your system tray menu.

3. Launch the <a href="{{ links.website.home }}">Pieces for Developers | Desktop App</a>

    If you've already installed the <a href="{{ links.website.pfd_desktop_install }}">Pieces for Developers | Desktop App</a> and it's having trouble connecting to Pieces for Developers | OS Server, try these quick <a href="{{ links.internal.faqs }}">Troubleshooting Solutions</a>.
