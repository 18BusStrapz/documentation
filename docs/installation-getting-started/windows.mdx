## Steps to Install
1. Download the Pieces for Developers <a target="_blank" href="{{ links.installation.pieces_suite_manager }}">Installation Manager</a> and proceed with the installation.

    This will install the Pieces for Developers Desktop App and Pieces OS.

2. Make sure Pieces OS Server is running in the background.

    Following the installation, ensure that the Pieces logo appears in your system tray menu.

3. Launch the Pieces for Developers Desktop App

    If you've already installed the Pieces Desktop App and it's having trouble connecting to Pieces OS, try these quick <a target="_blank" href="{{ links.internal.faqs }}">Troubleshooting Solutions</a>.

## Uninstalling Pieces for Developers 
To uninstall Pieces OS and the Pieces desktop app: 

1. Close both applications
2. Navigate to Settings > Apps > Installed Apps and locate Pieces/Pieces OS
3. Click on the three-dot dropdown menu and select "Uninstall"
