# Getting Started with Pieces for Developers on Windows
<ReleasePill>Compatible with Windows 10 Version 1809 or higher</ReleasePill>

![](/assets/installation/robots_windows.png)

<InstallButton>Download Pieces Suite Installer</InstallButton>

## Steps to Install
1. Download the Pieces for Developers <a target="_blank" href="{{ links.installation.pieces_suite_manager }}">Installation Manager</a> and proceed with the installation.

    - This will install the Pieces for Developers | Desktop App as well as Pieces for Developers | OS Server.

2. Make sure **Pieces for Developers | OS Server** is running in the background.

    - Following the installation, ensure that the Pieces logo appears in your system tray menu.

3. Launch the Pieces for Developers | Desktop App

    If you've already installed the Pieces for Developers | Desktop App, and it's having trouble connecting to Pieces for Developers | OS Server, try these quick <a target="_blank" href="{{ links.internal.faqs }}">Troubleshooting Solutions</a>.
