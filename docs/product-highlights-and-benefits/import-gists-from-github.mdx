- ### Import Gists from GitHub
Import saved code from GitHub and discover other users' Gists from inside the Pieces for Developers | Desktop App. When you import Gists, Pieces attaches metadata from its original repo along with PRs, related people, and additional context.

> You may be wondering if you can **create Gists** - read more about [Creating Gists from the Desktop App]
